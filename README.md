# ShivCosmic Portfolio

A premium, high-performance portfolio website for Shiv Cosmic Energy Solutions, featuring Astro Mapping™ and Vastu consultation services.

## Features
- **Modern UI/UX**: Built with Next.js 16, Tailwind CSS, and Framer Motion.
- **Performance**: Optimized with Lenis smooth scroll, reduced motion support, and efficient shader rendering.
- **Global Deployment**: Configured for static export (Hostinger/GitHub Pages).
- **Mobile First**: Fully responsive design with touch-optimized controls.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion, GSAP, Three.js (@paper-design/shaders-react)
- **Icons**: Lucide React
